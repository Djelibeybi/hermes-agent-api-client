tests, tag, and publication of `hermes-agent-api-client` v0.1.0. It is not a
pre-implementation plan and does not claim that GSD planned or executed the
historical implementation.

## Imported Evidence

The completed import recorded:

- The immutable public API and implementation at signed tag `v0.1.0`.
- The mapping from all eight completed requirements to tagged source and test

- The package boundary and historical context in `01-CONTEXT.md`.
